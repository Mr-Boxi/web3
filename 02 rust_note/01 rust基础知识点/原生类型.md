## 1.1 原生类型

### 1.1.1 标量类型

- 有符号整数: i8
- 无符号整数: u8,
- 浮点数: f32, f64.
- 字符: char.
- 布尔类型：bool.
- 单元类型： （）就是空元组。

### 1.1.2 复合类型

- 数组（array）: 如 [1,2,3]
- 元组（tuple） : 如（1，true）

## 1.2 自定义类型

### 1.2.1 结构体

- 元组结构体。 就是具名元组而已
- 经典的C风格结构体。
- 单元结构体。 不带任何字段。

### 1.2.2 枚举

任何一个在 `struct` 中合法的取值在 `enum` 中也合法。

## 1.5 语句与表达式

```rust
fn add_with_extra(x: i32, y: i32) -> i32 {
    let x = x + 1; // 语句
    let y = y + 1; // 语句
    x + y  // 表达式
}
```

- 语句与表达式是的区别就是表达式会返回值
