### web3js 什么
在进行网站或客户端开发时，就需要用到[web3-connect-network.js](https://github.com/ethereum/web3-connect-network.js/)。web3.js库是一个javascript库，可以执行很多与区块链进行交互的任务


### web3js安装

```bash
# 全局安装
npm install -g web3 
```
