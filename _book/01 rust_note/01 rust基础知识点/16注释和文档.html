代码注释

文档注释

  • 文档行注释 ////
/// 使用   `/// ` 结合markdown 使用
/// `add_one` 加 1 操作
///
/// # Examples
///
///

/// let age = 5; /// letn answer =add_one(age); /// assert_eq!(6, answer); /// ``` pub fn add_one(x: i32) -> i32 { x + 1 }


- 文档块注释  /**  ...  */

````rust
/** `add_one` 加 1 操作

let arg = 5; let answer = my_crate::add_two(arg);

assert_eq!(7, answer);

*/
pub fn add_one(x: i32) -> i32 {
    x + 1
}
`

块注释：/！ ... /

里面包含了compute等有用模块*/
